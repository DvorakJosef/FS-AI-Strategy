Sovereign AI for Finshape

Hybrid architecture where sovereign open-weight models (Qwen3-Coder, DeepSeek V4, GLM-4.7) cover 70-80 % of daily coding routine in EU jurisdiction, while Claude Opus and GPT-5 remain for the hardest 20-30 % of tasks.

~500 developers ~400 active AI users Banking software · DORA/GDPR compliance Current baseline full TCO ~$360-470k/year Senior CZ FTE ~$95-130k/year (fully loaded)
v4 update (2026-05-14): This version includes the full TCO incorporating human cost (FTE, harness adaptation, productivity tax). v3 only counted infrastructure cost and claimed "savings of $130-200k/year today" — that held for infra only. After factoring in 1-2 FTE platform/MLOps (CZ rates ~$95-130k/year fully loaded), harness adaptation ($45-115k one-time) and transition productivity tax: Scenario A is roughly cost-equivalent to baseline in year 1; year 2+ delivers savings of ~$25-195k/year (mid ~$110k), with ROI break-even at 12-24 months. Strategic value (sovereignty, DORA, vendor independence) remains the primary argument; pure cost savings is a secondary benefit.

Risks and mitigations

Strategic flexibility: model weights are portable, the gateway is self-owned, hybrid = graceful degradation.

  • Harness adaptation fails (new in v4) Pilot validates. Backup: keep Claude Code for 30 % of traffic, sovereign for 70 % via gateway.
  • CZ MLOps hiring delay (new in v4) External consultants $85-130/h as a bridge. Start hiring in Q3 2026 (Phase 0); timeline 6-9 months.
  • Productivity tax worse than estimated (new in v4) Champions program, slower rollout (team-by-team), preserved Claude Code fallback.
  • Anthropic/OpenAI raise prices 2-3× Reduce escalation, strengthen GLM-4.7 / Kimi K2.6 as near-frontier.
  • Nebius / EU API raises prices Migrate to Scenario B — model weights are portable between providers within weeks.
  • EU AI Act bans Chinese models Switch primary to Llama 4-Coder + Codestral. SWE-Bench degradation 5-10 points.
  • Provider outage Multi-region: primary Nebius FI, secondary OVHcloud FR. Gateway routes automatically.
  • Developers reject quality Pilot exposes this early. Claude Code Max remains in gateway as fallback (degradation, not replacement).
