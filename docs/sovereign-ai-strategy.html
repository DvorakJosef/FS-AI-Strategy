Rizika a mitigace

Strategická flexibilita: váhy modelů portovatelné, gateway vlastní, hybrid = graceful degradation.

  • Harness adaptace selže (nové v4) Pilot ověří. Backup: zůstat na Claude Code pro 30 % traffic, sovereign jen pro 70 % via gateway.
  • CZ MLOps hiring delay (nové v4) Externí konzultanti $85-130/h jako bridge. Začít hiring v Q3 2026 (Fáze 0), timeline 6-9 měs.
  • Productivity tax horší než odhad (nové v4) Champions program, pomalejší rollout po týmech, preserved Claude Code fallback.
  • Anthropic/OpenAI zdraží 2-3× Snížit eskalaci, posílit GLM-4.7 / Kimi K2.6 jako near-frontier.
  • Nebius / EU API zdraží Migrace na Scénář B — váhy modelů portovatelné mezi providery za týdny.
  • EU AI Act zakáže čínské modely Přepnout primární na Llama 4-Coder + Codestral. Degradace SWE-Bench 5-10 bodů.
  • Provider výpadek Multi-region: primary Nebius FI, secondary OVHcloud FR. Gateway routuje automaticky.
  • Vývojáři nepřijmou kvalitu Pilot odhalí dřív. Claude Code Max zůstává v gatewayi jako fallback (degradace, ne nahrazení).
